Troubleshooting

Notifications not working?

macOS

Notifications not appearing?

  1. Open System Settings
  2. Go to Notifications
  3. Find Google Chrome in the list
  4. Enable Allow Notifications
  5. Make sure Focus Mode is off

Windows

Notifications not appearing?

  1. Open Settings (Win + I)
  2. Go to System â†’ Notifications
  3. Make sure notifications are On
  4. Scroll down and find Google Chrome
  5. Enable notifications for Chrome
  6. Disable Focus Assist if active

Still having issues? Make sure Chrome is updated to the latest version and try restarting your browser.
